Leaflet EdgeMarker Plugin

Leaflet EdgeMarker is a Leaflet plugin which allows you to indicate Markers, Circles and CircleMarkers that are outside of the current view by displaying CircleMarkers at the edges of the map.

Usage

// create a map
var map = L.map('map').setView([51.905, 7.69], 10);

// add an OpenStreetMap tile layer
L.tileLayer('http://{s}.tile.osm.org/{z}/{x}/{y}.png', {
    attribution: '&copy; <a href="http://osm.org/copyright">OpenStreetMap</a> contributors'
}).addTo(map);


// add your Markers, Circles and CircleMarkers
L.circle([51.957807388715544, 7.591552734375], 1200).addTo(map);
L.circle([51.66339895213779, 9.378890991210938], 2200).addTo(map);
L.circleMarker([52.3688917060255, 9.722900390625]).addTo(map);
L.circleMarker([51.508742458803326, 9.942626953125]).addTo(map);
L.marker([53.553362785528094, 9.986572265624998]).addTo(map);
L.marker([52.51622086393074, 13.392333984375]).addTo(map);

// add the EdgeMarker to the map. It takes the options of a standard CircleMarker
L.edgeMarker({'radius':20}).addTo(map);
